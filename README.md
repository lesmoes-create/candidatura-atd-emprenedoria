## Publicació a GitHub Pages

1. Crea un repositori nou a GitHub (per exemple `candidatura-atd`).
2. Puja tots els fitxers d'aquesta carpeta (`index.html`, `README.md`, `docs/`, `img/`) a l'arrel del repositori.
3. Ves a `Settings → Pages` del repositori.
4. A `Source`, selecciona `Deploy from a branch`, branca `main`, carpeta `/ (root)`. Desa.
5. En uns minuts el lloc estarà disponible a `https://<usuari>.github.io/candidatura-atd/`.
